# Guide: On-Chain

The On-Chain page is a live feed of blockchain activity — large transfers, exchange
flows, smart-money moves, liquidations, and security events — so you can see what's
happening on-chain in real time. This guide explains it.

> Note: This is on-chain information, not a recommendation to buy or sell.

## What it shows

A continuously updating list of on-chain **alerts**. Each alert is a notable event
(e.g. a big transfer, a smart-money buy, a liquidation) with the coin, amount in USD,
and time.

## Alert type filters

Filter the feed by event type:
- **All** — everything.
- **Transfer** — token transfers between wallets.
- **Whale** — unusually large transfers (big holders moving funds).
- **Smart Money** — activity from wallets tracked as sophisticated/profitable.
- **Mint/Burn** — new tokens created (mint) or removed from supply (burn).
- **Buy / Sell** — on-chain buys and sells.
- **Deposit** — funds moving into exchanges (often a sign of intent to sell).
- **Liquidation** — leveraged positions force-closed.
- **Position** — position opens/changes.
- **Security** — security-related events (exploits, suspicious activity).

## Amount filter

Filter by transaction size: **All, $10K+, $100K+, $1M+, $10M+**. Use bigger thresholds
to focus only on market-moving flows.

## Stats

- **Total Alerts** — how many alerts, with the count in the last 24h.
- **Last Hour** — recent activity level (Active flow vs Quiet).
- **Top Activity** — the most active category/coin right now.
- **Whale threshold** — the "whale" size is dynamic (computed from recent alert sizes,
  roughly the 95th percentile), so it adapts to current market conditions.

## How to read it

- **Exchange deposits / inflows** can hint at selling pressure; **withdrawals** can hint
  at accumulation.
- **Whale + Smart Money** buys are worth noting; large **Sell/Deposit** spikes can
  precede volatility.
- Raise the **amount filter** to cut noise and see only big money.

## Important note

On-chain flows are context, not signals to act on directly, and not financial advice.
Trading decisions are your own.
